# atividade_final_senacfull2026
Mini Sistema Bancário em JavaScript


Este projeto foi desenvolvido como atividade final com o objetivo de simular operações básicas de uma conta corrente utilizando lógica simples em JavaScript e HTML.

O sistema permite ao usuário realizar depósitos, efetuar saques e consultar o saldo disponível, garantindo que as operações sejam processadas corretamente.

Também foram aplicadas validações importantes, como impedir saques com valor superior ao saldo disponível e bloquear a inserção de valores inválidos, como números negativos ou vazios.

Além disso, foi utilizado um array para armazenar o extrato das movimentações, junto com estruturas de repetição como "for" e "while", estruturas condicionais "if" e "else", botões simples em HTML e exibição de informações no "console.log".

Funcionalidades

- Depositar valores
- Realizar saques
- Consultar saldo atual
- Visualizar extrato de operações
- Validação de saldo insuficiente
- Validação de valores inválidos
- Histórico das movimentações com array

---

Tecnologias utilizadas

- HTML5
- JavaScript
- Lógica de programação
- Estruturas condicionais ("if", "else")
- Estruturas de repetição ("for", "while")
- Arrays
- "console.log"

---

Objetivo da atividade

Desenvolver um minissistema bancário simples que simule operações do dia a dia de uma conta corrente, reforçando os conceitos fundamentais da programação com JavaScript.

---

Como executar

1. Baixe os arquivos do projeto
2. Abra o arquivo "index.html" no navegador
3. Digite um valor no campo disponível
4. Utilize os botões para depositar, sacar ou consultar o extrato
5. Abra o console do navegador ("F12") para visualizar o extrato e mensagens do sistema
